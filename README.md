# SunCart 
this is a website where you can explore products as a user  purchase seasonal products like sunglasses, summer outfits, skincare, beach accessories, and more.

* Live link : https://suncart-7eim.vercel.app

# Key Features 
## Betterauth
* This website has auth built in with betterauth so that users can explore this website as users with MongoDB
## Log in with google
* I know google log in isn't something special but its an feature of a mordern website
## log in with email and password
* people can register this website with email and password and other stuff
## animation
* there is a little touch from animate.css
## protected route
* to access the details page users must log in or register if they don't have an account

# Tecnology stack 
* Nextjs
* Betterauth
* Tailwind css
* daisyUI and  HereUi
* proxy js
